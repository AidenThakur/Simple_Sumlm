# Simple_Sumlm
A simple summary lm project that will take text files and create a simple summary based off the text provided. The goal of this project was to teach myself how to start small a small language model and understand how to train it and how to look and fine tune the projects so that it will provide better generation. I also wanted to learn more about perplexity and other metrics when evaluating models.
This project is a small that is a work in progress.
